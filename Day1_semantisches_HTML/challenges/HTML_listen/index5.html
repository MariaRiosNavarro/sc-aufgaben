Was sind Listen

In HTML sind Listen strukturierte Elemente, die verwendet werden, um Informationen in geordneter oder ungeordneter Form darzustellen. Es gibt drei Hauptarten von Listen: ungeordnete Listen (unordered lists), geordnete Listen (ordered lists) und Definitionslisten (definition lists).

Unordered List

Eine ungeordnete Liste wird verwendet, um Elemente ohne spezifische Reihenfolge darzustellen. Die einzelnen Listenelemente werden typischerweise durch Aufz√§hlungszeichen, Punkte oder andere Symbole markiert.

In HTML wird eine ungeordnete Liste mit dem <ul> Tag erstellt, und jedes Listenelement wird mit dem <li>-Tag definiert. Sowohl <ul> als <li> brauchen en end tag: 

        <ul>
            <li>Element 1</li>
            <li>Element 2</li>
            <li>Element 3</li>
        </ul>

Ordered List

Eine geordnete Liste wird verwendet, um Elemente in einer bestimmten Reihenfolge darzustellen. Die Listenelemente werden normalerweise numerisch (1, 2, 3, ...) oder alphabetisch (a, b, c, ...) nummeriert.

In HTML wird eine ungeordnete Liste mit dem <ol> Tag erstellt, und jedes Listenelement wird mit dem <li>-Tag definiert. Sowohl <ol> als <li> brauchen en end tag: 

        <ol>
            <li>Element 1</li>
            <li>Element 2</li>
            <li>Element 3</li>
        </ol>
